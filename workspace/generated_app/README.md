# DeliveryFlow Tasks

Application de demonstration generee pour Demo Client.

## Structure

- `backend/` : API FastAPI
- `frontend/` : interface React

## Statut

Cette application est un artefact généré par le repo principal.

Elle sert aujourd'hui de démonstrateur de génération, mais n'est pas encore totalement finalisée :

- backend encore léger
- persistance SQLite à renforcer
- frontend encore peu branché sur un backend réel

## Intention cible

Le workspace généré doit converger vers :

- un backend FastAPI avec CRUD de tâches
- une persistance SQLite réelle
- un frontend React/Vite/Tailwind connecté au backend
- une expérience de démonstration crédible pour le hackathon
